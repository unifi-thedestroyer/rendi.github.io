Tata Cara Sewa Motor Solo

Syarat Sewa Motor Solo di Sahaja rent., Silahkan dipilih minimal 3 id yang di jaminkan. Boleh di gabung dengan temannya.

  • E-Ktp + SIM
  • Kartu Keluarga
  • NPWP / Billing Listrik atau Air
  • Foto Tiket Pesawat / Kereta Api / Bus
  • Foto Voucher Hotel / Homestay
  • Foto Kartu Kredit / Debit (Nomer Ditutup)
  • Foto Kartu Nama Perusahaan / Mahasiswa
  • Nama Account Linked / Instagram / Twitter
  • Foto Rekening Tabungan atas Nama
  • Wajib transfer ke rekening account kami sesuai dengan nama penyewa.

Langkah Sewa Motor di Solo dengan Sahaja Rental adalah sebagai berikut :

  • Cek Harga Sewa Motor Solo di Web kami sesuai dengan kebutuhan.
  • Silahkan lakukan pemesanan via Form, Chat Wa (Jangan Lewat Telp)
  • Setelah kami konfirmasi ketersediaan unit motor, Silahkan Kirim data
  • Kirimkan data persyaratan yang kami minta via chat wa
  • Kirimkan data lokasi serta jam pengantaran yang akan di inginkan.
  • Silahkan lakukan Proses DP sesuai dengan atas nama penyewa ke rekening kami.
  • Bookingan anda kami konfirmasi dan 100 % Aman
  • Saat serah terima kendaraan, Tim Validasi kami akan melakukan proses cek data kelengkapan anda. (Asli / Palsu)
  • Jika data anda sudah valid sesuai dengan reservasi awal. Langsung Proses serah terima motor di lokasi yang anda inginkan.
  • Jika sudah selesai, tim delivery kami akan mengambil kembali motor yang anda sewa.
  • Jangan lupa daftar member untuk mendapatkan discount harga langganan.

PERINGATAN : Jika ada kami temukan pelanggaran terhadap pemalsuan identitas, maka kami tidak akan segan segan mengantarkan anda ke pihak yang berwajib untuk di proses hukum. Dalam hal ini kami tidak akan pernah ada toleransi.
